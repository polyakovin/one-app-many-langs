# [Kotlin](https://kotlinlang.org/)

## Installation

Download and install [Kotlin](https://kotlinlang.org/docs/tutorials/command-line.html).

Compile the code
```sh
kotlinc sort.kt -include-runtime -d sort.jar
```

## Launch

```sh
java -jar sort.jar 0 24 3 1 -5 1 1.1
```
